# 彩票出手策略测试说明

## 功能概述
实现了完整的MainForm功能测试用例，包含：

1. 每次生成一组纯随机的350注号码
2. 执行出手判断和是否中奖判断
3. 不中奖就继续出手，最多连续8次不中为爆掉一个计划
4. 可以设置循环执行步骤1和2多少次
5. 统计爆掉次数和中的次数
6. 最后统计一共中了多少次，爆了多少次

## 测试组件

### TestMainFormFunctionality类
- 主要测试逻辑实现在`Tests\TestMainFormFunctionality.cs`
- 提供了完整的出手策略测试流程
- 包含号码生成、出手判断、中奖判断等功能

### 运行方式
- 编译：`dotnet build ConsoleTestApp.csproj`
- 运行：`dotnet run --project ConsoleTestApp.csproj`

## 测试特点
- 使用TXFFC.txt文件中的真实开奖数据进行测试
- 每轮测试基于真实的历史数据序列
- 实现了完整的出手周期逻辑
- 提供详细的统计报告

## 输出信息
- 每轮测试的详细结果
- 总体统计结果（总中奖次数、总爆掉次数等）
- 平均值统计

此测试系统可用于验证不同的出手策略效果。
